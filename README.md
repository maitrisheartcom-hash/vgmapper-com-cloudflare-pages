# VGMapper

Static Cloudflare Pages restore for `vgmapper.com`.

## Deploy

- Build command: none
- Output directory: `/`
- Runtime: static files only

Do not add Cloudflare Workers, Pages Functions, `_worker.js`, `functions/`, or `wrangler.toml` to this repository.

The historical pages were restored from public Wayback captures before 2013. Update by committing and pushing this repository; Cloudflare Pages should be connected through Git.
